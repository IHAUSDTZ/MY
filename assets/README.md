# IHAUS Project Assets

This directory contains brand assets and sample royalty-free images for the Integrated Hill Areas Upliftment Society (IHAUS).

## Contents
- `hero-rural-manipur.jpg`: Sample high-res hero image for the homepage.
- `logo-ihaus-vector.svg`: NGO logo (placeholder).
- `program-livelihood-sample.jpg`: Representative image for agricultural programs.
- `impact-road-before-after.jpg`: Case study visual assets.

## Usage
These assets should be moved to the `/public` folder for production use or hosted on a CDN.
Always use `next/image` with `placeholder="blur"` for optimal performance.
